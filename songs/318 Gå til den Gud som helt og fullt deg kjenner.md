# 318 Gå til den Gud som helt og fullt deg kjenner

Gå til den Gud som helt og fullt deg kjenner!
Gå til den Gud som vet når angsten brenner!
Gå til den Gud som er mer nær enn venner!
Han tar imot deg.

Stol på din Gud, som aldri, aldri svikter!
Stol på din Gud, og bort din uro viker!
Stol på din Gud, som aldri, aldri sviker!
Han tar imot deg.

Hvil deg i Gud når verden rundt deg larmer!
Hvil deg i Gud som seg så helt forbarmer!
Hvil deg i Gud som barn i moders armer!
Han tar imot deg.
