# 135 Å kom nå heile heimsens hær

Å kom nå, heile heimsens hær, Og heidra Gud som skapte oss!
Ja, lov han, all den vide verd, For alt, og mest for Kristi kross!

Hans signing er som sol om kveld, Som dogg når dagen gryr i aust,
Som våren på dei fagre fjell, Som frukta i den klåre haust.

Gud, auk ditt lys i hugen vår Så vi din vilje kjenner grant!
Ditt ord er oss ei stjerne klår, Vi vitnar det er evig sant.
